## What's in This Repo

[`garage-ai-cheatsheet.md`](./garage-ai-cheatsheet.md) — every install command, config snippet, and script from the book, organized by chapter. Copy and paste directly into your terminal without retyping anything from the printed page.

Covers:
- Homebrew and macOS setup
- Ollama: install, pull, run, manage models, background service, API
- MCP: filesystem, memory, and search servers, Open WebUI, config file
- RAG: Chroma setup, `index_docs.py`, `ask_rag.py`, PDF/Word support
- Agents: Open Interpreter, LangChain, CrewAI, daily briefing script, LaunchAgent plist
